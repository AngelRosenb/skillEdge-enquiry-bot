# SkillEdge Enquiry Bot 🎓

An AI-powered Educational Enquiry Assistant that simulates 
WhatsApp-based interaction for training academies.

## Features
- Course enquiry handling
- Lead capture (name, phone, qualification)
- Enrollment flow
- Off-topic redirection

## Tech Stack
- Python + Flask
- Groq LLM (llama-3.3-70b)
- HTML/CSS (WhatsApp UI)

## How to Run
1. Clone the repo
2. Install dependencies: pip install flask groq python-dotenv
3. Add your GROQ_API_KEY to .env file
4. Run: python app.py
5. Open: http://127.0.0.1:5000
